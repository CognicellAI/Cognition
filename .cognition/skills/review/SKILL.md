name: code-review
description: Review code for quality, security, and best practices.
---

To perform a code review:
1. Read the changed files carefully.
2. Check for potential security vulnerabilities (SQL injection, XSS, etc.).
3. Evaluate code quality, naming conventions, and architectural alignment.
4. Suggest optimizations for performance or readability.
5. Provide a summary of findings and specific improvement suggestions.
